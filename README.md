# Sentiment Analysis App

This is a simple and interactive web application for sentiment analysis of social media comments. The app predicts whether a given comment expresses a positive, negative, or neutral sentiment.

## Features

- Text input for any social media comment.
- Real-time sentiment prediction using a trained machine learning model.
- Interactive and user-friendly interface built with Streamlit.
- Model trained using Logistic Regression on labeled sentiment data.
- Text preprocessing using TF-IDF vectorization.
